# New Portfolio

Modern personal portfolio website for Muhammad Rizky Suryanata with a cosmic / galaxy / futuristic theme.

## Tech Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
- Vercel deployment

## Features

- Futuristic cosmic background with stars and meteors
- Responsive layout for desktop, tablet, and mobile
- Hero section with AI demo video showcase
- Surya OS personal profile section
- Interactive terminal / AI ops console
- Skills, projects, experience, education, testimonials, and contact sections
- Project detail pages:
  - `/projects/mlenz`
  - `/projects/mia`
  - `/projects/mzone`
- ID / ENG language toggle
- Music popup with looping audio

## Getting Started

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```txt
http://localhost:3000
```

## Build

```bash
npm run build
```

## Lint

```bash
npm run lint
```

## Deploy

This project is ready to deploy on Vercel.

```bash
npx vercel --prod
```
